中文AI交互规则 项目说明 本项目为个人学习探索阶段的初步研究文档，仅围绕日常AI工具使用、数据开发交互过程中遇到的实际问题，梳理中文语境下AI交互的基础通用规则与思路，无成熟产品、无完整落地代码，仅作个人思考记录与学习交流。 核心解决痛点 1. 日常使用AI进行编程、数据处理、需求交互时，中文表述歧义性强，相同意图不同表述易导致AI输出偏差，沟通成本高； 2. 无统一的基础交互规范，个人学习、简单开发场景下，难以高效传递自身需求，AI执行结果与预期不符； 3. 零基础/入门阶段学习者，难以用精准语言和AI形成有效配合，影响学习与实操效率。   已公开内容 仅包含中文AI交互基础通用规范，为初步梳理的基础交互原则，无核心设计、无具体实现代码、无完整体系落地细则，后续仅根据个人学习进度做基础补充，不涉及宏大框架与版本迭代规划。 备注 本项目纯属个人学习总结，非商用、非成熟方案，仅供交流学习，请勿直接用于生产环境与项目落地
